Accessible Documents Overview

Updated: 2025-11-20 · Next review: 2026-02-20

Summary

Most accessibility issues in documents are avoidable when you use styles, alt text, descriptive links, and accessible exports. This overview connects the dots between Word, Google Docs, PDFs, and slides.

Core practices

  • Use built-in heading styles instead of manual font changes.
  • Provide alt text or mark decorative images as such.
  • Use real lists and tables, not manually spaced text.
  • Export to tagged PDF only after the source is accessible.
