Browser Extensions (Automated Testing)

Quick scans that catch 30-40% of accessibility issues. Essential first step, but not comprehensive.

Tool Price Best For Pros Cons
WAVE
wave.webaim.org		 Free Beginners, content authors Visual overlay shows issues in context; explains errors clearly; works offline Can be overwhelming on complex pages; no CI/CD integration
axe DevTools
deque.com/axe		 Free(Pro version paid) Developers, QA Industry standard engine; integrates with CI/CD; detailed remediation guidance Pro features cost money; less visual than WAVE
Accessibility Insights
accessibilityinsights.io		 Free Manual + automated testing Guided assessments; FastPass for quick checks; Tab stops visualizer Chrome/Edge only; more complex UI
Lighthouse
Built into Chrome		 Free Developers (quick check) Built into Chrome DevTools; includes performance + SEO; CI/CD friendly Less comprehensive than axe; fewer explanations
ANDI
ssa.gov/accessibility/andi		 Free Government/508 testing Bookmarklet works anywhere; focuses on 508; shows accessible name calculation Dated interface; less intuitive

Our recommendation

Start with WAVEfor visual feedback and learning.Add axe DevToolsif you're a developer integrating into your workflow. UseAccessibility Insightswhen you need guided manual testing.

Screen Readers

Manual testing with screen readers catches issues automation misses. Test with at least one.

JAWS

TalkBack
Built into Android		 Android Free Chrome Built into Android. Essential for Android app testing.

Windows:Install NVDA (free) and test with Firefox or Chrome.Mac:Use built-in VoiceOver with Safari.Mobile:Use VoiceOver (iOS) or TalkBack (Android) with the default browser.

Recommended Testing Approach

No single tool catches everything. Use this combination:

  1. Automated scan:WAVE or axe DevTools (catches ~30-40% of issues)
  2. Keyboard testing:Tab through the page manually
  3. Screen reader:NVDA or VoiceOver (catches semantic issues)
  4. Visual check:Zoom to 200%, check contrast, color-only indicators
  5. Enterprise scan:SiteImprove/DubBot for ongoing monitoring
