Keyboard Navigation Essentials

Essentials

  • All interactive elements must be reachable via Tab/Shift+Tab in a logical order.
  • Provide a visible focus indicator with sufficient contrast; do not remove outlines without a replacement.
  • Links and buttons must activate on Enter/Space.
  • No keyboard traps: users can leave dialogs and widgets with keyboard only.
  • Include a Skip to main content link as the first focusable element.
