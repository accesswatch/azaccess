Accessibility Statement

Our commitment to making this site accessible to everyone.

Our commitment

The University of Arizona Digital Accessibility team is committed to ensuring this resource site is accessible to the widest possible audience, including people with disabilities.

We hold ourselves to the same standards we ask of others:WCAG 2.2 Level AA.

If we're going to help Arizona achieve accessibility, our own site must lead by example.

Measures we take

We maintain accessibility by:

  • Using semantic HTML and proper heading structure
  • Providing text alternatives for images and media
  • Ensuring keyboard accessibility throughout the site
  • Maintaining sufficient color contrast (minimum 4.5:1 for text)
  • Including visible focus indicators for keyboard users
  • Providing skip links to bypass navigation
  • Testing with screen readers (JAWS, NVDA, VoiceOver)
  • Running automated scans with axe, WAVE, and Accessibility Insights
  • Conducting manual keyboard-only testing before each release

Conformance status

Compatibility with assistive technology

This site is designed to be compatible with:

  • Screen readers:JAWS, NVDA (Windows), VoiceOver (macOS/iOS), TalkBack (Android)
  • Screen magnifiers:ZoomText, Windows Magnifier, macOS Zoom
  • Speech recognition:Dragon NaturallySpeaking, Windows Speech Recognition
  • Keyboard-only navigation:All functionality is available without a mouse

We test primarily with:

  • Chrome + NVDA on Windows
  • Safari + VoiceOver on macOS
  • Firefox + JAWS on Windows
  • Safari + VoiceOver on iOS

Accessibility features of this site

Navigation

  • Skip link to bypass navigation
  • Consistent navigation on every page
  • Breadcrumb trails showing location
  • Clear, descriptive link text

Structure

  • Proper heading hierarchy (H1-H6)
  • ARIA landmarks for page regions
  • Lists marked up as lists
  • Data tables with headers

Visual

  • Text can be resized to 200% without loss
  • Color is not the only indicator
  • Minimum 4.5:1 contrast ratio
  • Respects prefers-reduced-motion

Interaction

  • All controls keyboard accessible
  • Visible focus indicators
  • Form fields have labels
  • Error messages are clear

Feedback and contact

We welcome your feedback on the accessibility of this site. Please let us know if you encounter barriers or have suggestions for improvement.

To report an accessibility issue:

  • Email:accessibility@arizona.edu
  • Phone:520-621-3268
  • Form:Access Report

When reporting an issue, please include:

  • The page URL where you encountered the problem
  • What you were trying to do
  • What assistive technology you were using (if any)
  • What happened vs. what you expected

We aim to respond to accessibility feedback within2 business days.

Formal complaints

If you are not satisfied with our response, you may file a formal complaint through:

  • University of Arizona Office of Institutional Equity
  • U.S. Department of Justice ADA Complaint Portal
  • U.S. Department of Education Office for Civil Rights
