For Web Developers

If you manage a site that links to accessibility.arizona.edu, use these patterns to update your links:

Recommended Link Format

Always use the full path for stability:

https://accessibility.arizona.edu/docs/[page-name].html

Server Redirect Rules

If implementing redirects, use 301 (permanent) redirects for SEO:

# Apache .htaccess example
Redirect 301 /getting-started /docs/accessibility-101.html
Redirect 301 /word-guide /docs/doc-word.html
