Accessible Social Media Guide

Make your social posts accessible to everyone‚Äîincluding people using screen readers.

Universal Tips (All Platforms)

Write Alt Text for Images

Describe what's IN the image, not just "photo" or "graphic".

"Event flyer"
"Flyer for Spring Career Fair, March 15, 10am-2pm at Student Union. Features UA Wildcat logo."

Use CamelCase for Hashtags

Capitalize each word so screen readers pronounce them correctly.

#accessibilitymatters(reads as one long word)
#AccessibilityMatters(reads as "Accessibility Matters")

Limit Emoji Use

Screen readers read each emoji aloud. 5 clapping emojis = "clapping hands clapping hands clapping hands..."

"Great job! "
"Great job! "

Caption All Videos

Auto-captions are a start but need editing. 85% of social video is watched without sound!

Platform-Specific Guides

ùïè / Twitter Accessibility

Quick Reference

  • Add alt text:When composing, click image "Add description" (or ALT button)
  • Character limit:Alt text up to 1,000 characters
  • Auto-captions:Available for videos

Adding Alt Text to Images

  1. Compose your tweet and add an image
  2. Click the image thumbnail
  3. Click "Add description" or the "ALT" badge
  4. Write your description (up to 1,000 characters)
  5. Click "Save"

Pro tip:Enable "Image descriptions reminder" in Settings Accessibility to get prompted.

Thread Accessibility

  • Number your tweets: "1/5", "2/5" etc.
  • Start with context‚Äîdon't assume people saw tweet #1
  • Put alt text on every image in the thread

Instagram Accessibility

Quick Reference

  • Add alt text:Before posting Advanced Settings Write Alt Text
  • Edit existing:‚Ä¢‚Ä¢‚Ä¢ menu Edit Alt text
  • Stories:No native alt text‚Äîdescribe in caption or use voiceover
  • Reels:Add captions via the captions sticker or auto-captions

Adding Alt Text to Posts

  1. Create your post as usual
  2. On the caption screen, tap "Advanced Settings" at the bottom
  3. Tap "Write Alt Text"
  4. Describe each image (swipe for multiple)
  5. Tap "Done" then share

Note:Instagram auto-generates alt text, but it's generic ("may contain: 2 people, outdoor"). Custom is better.

Stories & Reels

  • Stories:Use the Captions sticker for any speaking
  • Reels:Tap "Captions" to auto-generate, then review
  • If showing text on screen, read it aloud too
  • Avoid flashing content (seizure risk)

Facebook Accessibility

Quick Reference

  • Add alt text:Click photo Edit Alt Text Override generated
  • Auto-captions:Available for videos
  • Live captions:Enable in Live Producer

Adding Alt Text to Photos

  1. Upload your photo
  2. Click "Edit" on the photo
  3. Click "Alt Text" in the left panel
  4. Select "Override generated alt text"
  5. Write your description
  6. Click "Save"

Facebook Live Accessibility

  • Enable auto-captions in Live Producer before going live
  • Speak clearly and at a moderate pace
  • Describe any visuals you're showing
  • Repeat questions from comments before answering

LinkedIn Accessibility

Quick Reference

  • Add alt text:Click image after upload "Add alt text"
  • Character limit:300 characters for alt text
  • Documents:Upload accessible PDFs, not image-only

Adding Alt Text

  1. Start a post and add an image
  2. Click "Alt text" below the image
  3. Write your description (300 char limit)
  4. Click "Save"

Document Posts

  • If sharing slides/documents, ensure the source file is accessible
  • Add a text summary in the post for key points
  • Don't rely solely on infographics‚Äîexplain the data in text

TikTok Accessibility

Quick Reference

  • Auto-captions:Available‚Äîtap "Captions" when editing
  • Text-to-speech:Can read on-screen text aloud
  • Photosensitivity:TikTok can warn for flashing content

Adding Captions

  1. Record or upload your video
  2. Tap "Captions" on the editing screen
  3. Wait for auto-generation
  4. Tap captions to edit any errors
  5. Position captions so they don't cover important content

Best Practices

  • Add captions even if using trending audio‚Äîviewers watch muted
  • If using text-to-speech, review the pronunciation
  • Describe visual gags or reactions verbally
  • Avoid rapid flashing or strobing effects
