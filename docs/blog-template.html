How to Write Effective Alt Text: A Complete Guide

Alt text (alternative text) is one of the most fundamental accessibility practices, yet it's often misunderstood. This guide covers everything you need to know about writing effective alt text for images.

What is Alt Text?

Alt text is a written description of an image that serves as an alternative when the image cannot be seen. It's stored in the HTMLaltattribute:

<img src="cat.jpg" alt="Orange tabby cat sleeping on a blue couch">

Screen readers announce alt text to users who are blind or have low vision. It also displays when images fail to load or when users have images disabled.

Why Alt Text Matters

  • Accessibility:Enables blind and low-vision users to understand image content
  • Legal compliance:Required by WCAG 1.1.1 (Level A)
  • SEO:Search engines use alt text to understand and index images
  • Usability:Helps all users when images don't load

How to Write Good Alt Text

1. Be Specific and Concise

Describe what's in the image, not around it. Aim for 125 characters or less when possible.

2. Consider Context

Alt text should convey the image's purpose in context. A photo of a cat might need different alt text in a pet adoption article versus an article about cat behavior.

3. Don't Start with "Image of" or "Picture of"

Screen readers already announce "image" or "graphic," so this is redundant.

4. Include Relevant Details

If text, colors, expressions, or actions are important to understanding the image, include them.

Examples: Good vs. Bad

Good Alt Text

  • "Student using laptop in library study area"
  • "Bar chart showing enrollment growth from 2020-2025, rising from 40,000 to 48,000 students"
  • "UA Wildcat mascot high-fiving fans at football game"

Poor Alt Text

  • "Image" or "Photo"
  • "IMG_2847.jpg"
  • "Picture of a student"
  • "Chart"

Special Cases

Decorative Images

If an image is purely decorative (doesn't convey information), use an empty alt attribute:

<img src="decorative-line.png" alt="">

Complex Images (Charts, Diagrams)

Provide a brief alt text plus a longer description nearby or linked:

<img src="enrollment-chart.png" 
 alt="Enrollment trend chart. Full data in table below.">

Images of Text

Include the exact text in the alt attribute. Better yet, use real text instead of images of text.

Linked Images

Describe the link destination, not the image itself:

<a href="homepage.html">
 <img src="logo.png" alt="University of Arizona home">
</a>

Tools to Help

  • WAVE- Flags missing alt text
  • axe DevTools- Checks alt text quality
  • Word and PowerPoint accessibility checkers
  • Alt Text Decision Tree

Key Takeaways

  • Every meaningful image needs alt text
  • Be specific, concise, and contextual
  • Don't start with "image of"
  • Use empty alt (alt="") for decorative images
  • Complex images need extended descriptions
