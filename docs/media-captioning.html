Media & Captioning Guide

Demo summary of the shared media-captioning guidance.

Decision flow

Use this simplified flow to decide when to use machine captions, when to request professional captions, and when to add audio description.

  1. Is the media required for a course, program, or employment? If yes, plan for accurate captions.
  2. Start with machine captions in Panopto or Zoom, then edit for accuracy.
  3. Request professional captions for high-stakes or widely reused content.
  4. Consider audio description when important visuals are not described in the narration.
