Brightspace Accessibility Checklist

Key steps

  1. Use HTML templates with semantic headings for content modules.
  2. Add alt text and provide transcripts/captions for images and media.
  3. Configure quizzes for extended time and flexible formats.
  4. Encourage descriptive discussion subjects and accessible media from students.
  5. Keep tables simple and set header rows/columns in the HTML editor.
  6. Use descriptive link text and validate LTI tool accessibility.
  7. Run the Brightspace Accessibility Checker and fix issues before publishing.
