Canva & Accessible Flyers

Owner: Digital Accessibility Team · Last reviewed: 2026-02-17 · Next review: 2026-05-17

What this page helps you do

  • Create flyers and graphics in Canva that are readable, shareable, and accessible to people using assistive technology.
  • Export and publish accessible alternatives (HTML/text/PDF) alongside image assets.
  • Apply quick checks that fix the most common barriers.

Immediate wins (5–15 minutes)

  • Use an approved template and keep important text as real, selectable text.
  • Provide a short accessible summary near the flyer when publishing.
  • Keep the editable source file so you can produce accessible exports later.

Checklist: Before you publish

  1. Structure & content — use real text, logical reading order, clear event details.
  2. Contrast & typography — meet minimum contrast, prefer larger readable fonts.
  3. Color and visual cues — don’t use color alone to convey key info.
  4. Images and icons — add concise alt text for exported images when used on web/email.
  5. File export — keep editable source; publish accessible HTML/text or accessible PDF alongside image exports.

Canva-specific tips

  • Verify contrast and swap templates that use low-contrast palettes.
  • Avoid placing important text over complex background imagery; use solid overlays.
  • Prefer single-column, top-to-bottom layouts for simple reading order.

Publishing patterns

  • Email: embed image + visible text summary + link to accessible page or attached accessible PDF.
  • Web: include `alt` text and a visible HTML summary; for event pages, prefer native HTML event details over images.
