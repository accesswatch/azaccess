Accessible Word & PowerPoint Basics

Updated: 2025-11-20 · Next review: 2026-02-20

Why it matters

Microsoft 365 files count as conventional electronic documents under Title II when they stay in active circulation. Fixing accessibility in Word and PowerPoint prevents repeated remediation work when exporting to PDF or posting in Brightspace.

Key actions

  1. Use built-in styles for headings, lists, and tables instead of manual formatting.
  2. Write clear alt text for informative images; mark decorative images as decorative.
  3. Check slide reading order in PowerPoint using the Selection Pane.
  4. Ensure sufficient color contrast for text and links.
  5. Name links descriptively (avoid "click here").
  6. Use real tables for data with header rows identified.
  7. Run the Accessibility Checker and resolve issues before publishing.
  8. Export to tagged PDF only after the source passes checks.

Quick checklist

  • Heading styles applied with no skipped levels.
  • Alt text present or images marked decorative.
  • Slide reading order verified.
  • Color contrast and link styling meet guidance.
  • Tables have header rows identified.
  • Accessibility Checker reports zero errors.
  • Document language set correctly.
