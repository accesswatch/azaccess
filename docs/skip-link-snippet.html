Skip Link Snippet

Standard pattern

Use this snippet to provide a visible, keyboard-accessible way to jump directly to the main content of a page.

<a class="skip-link" href="#maincontent">Skip to main content</a>

Ensure it is the first focusable element inside the <body> and that it points to the id of your main element.
