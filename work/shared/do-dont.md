```markdown
**Do**
- Use descriptive link text such as “Request captioning support.”
- Provide at least 5 business days for accommodation requests.
- Test media players with keyboard and screen readers.

**Don’t**
- Use “click here” or URLs as visible link text.
- Announce accommodations only via PDF attachments.
- Autoplay videos with sound without controls.
```

Copy the pattern and replace bullets per scenario.
