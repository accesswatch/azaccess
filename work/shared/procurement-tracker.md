Use this tracker to record procurement accessibility decisions, remediation SLAs, and exception justifications. Store a copy in your unit's procurement folder and attach to the contract record.

## Fields (suggested)

- Vendor

- Product / Module

- Procurement reference number

- VPAT date & version

- Scope (user/admin/features)

- Risk rating (Low / Medium / High)

- Remediation commitments (scope + dates)

- Acceptance status (Pending / Approved / Conditional / Rejected)

- Exception documented (Yes/No) — if Yes, include justification and review date

- Owner / Responsible party

- Last reviewed

---

## Usage

1. Populate during intake when VPAT/ACR is received.

2. Attach demo recording and axe/Lighthouse output for each entry.

3. Use the reviewer worksheet (procurement-reviewer.csv) to produce the procurement score and attach results.

4. Review exceptions quarterly and update acceptance status after remediation verification.
