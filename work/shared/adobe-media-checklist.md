- [ ] Document styles mapped to semantic tags (headings, lists)
- [ ] Images: alt text provided or marked decorative
- [ ] PDF: `Create Tagged PDF` enabled and Acrobat Full Check passed
- [ ] Reading order validated in Articles panel / Acrobat
- [ ] Captions: edited speech-to-text captions; sidecar SRT/VTT/TTML included
- [ ] Transcripts included for audio/video
- [ ] On-screen graphics meet contrast requirements
- [ ] Exported sample files attached to submission (PDF, SRT, MP4)
- [ ] Accessibility report (Acrobat / tool outputs) attached
