UA Accessibility Framework (Full Version)

This page presents the complete learning-path outline. Use the accordion version for condensed browsing or print this page for a checklist.

Understanding Digital Accessibility

This learning path introduces foundational digital accessibility concepts, legal frameworks, user needs, and the principles of inclusive design. It explains how disabilities intersect with technology, outlines assistive technologies, and helps learners understand barriers and solutions. (Paraphrased with attribution to LinkedIn Learning.)

Prepare for Digital Accessibility

This path focuses on planning accessibility from the beginning of a project. It covers accessible design thinking, content workflows, organizational readiness, and strategies for incorporating accessibility into documents, media, and digital products before creation begins. (Paraphrased with attribution to LinkedIn Learning.)

Build for Digital Accessibility

This learning path teaches hands-on techniques for creating accessible content—documents, PDFs, presentations, websites, and applications. It emphasizes WCAG principles, structure, semantic markup, color contrast, keyboard navigation, and testing practices. (Paraphrased with attribution to LinkedIn Learning.)
