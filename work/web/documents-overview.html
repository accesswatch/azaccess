Universal quick checks

Before publishing any document, verify these basics:

  1. Run the built-in checker â€” Word, PowerPoint, and Google Docs all have accessibility checkers. Use them.
  2. Check heading structure â€” Are you using Heading 1, Heading 2, etc. in logical order?
  3. Review images â€” Does every meaningful image have alt text?
  4. Test links â€” Do they describe where they go? Can you understand them without surrounding context?
  5. Verify contrast â€” Is all text readable against its background?
